# Phone BooK

## Description

Simple phone book application that let user add entries to the phone book and search for them.

## Features

1. Add entries.
2. search for entries.

## Technologies

1. React
2. Javascript
3. Html
4. Css

## How to the App Locally

1. git clone the repository
2. cd into folder
3. run "npm install"
4. run "npm run dev"
